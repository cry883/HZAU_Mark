# HZAU 校内评分社区

面向华中农业大学校园场景的垂直评分社区。  
项目以 Web 形态优先落地，支持校内师生对食堂、风景、教学楼、健身房等内容进行榜单化评价，并通过审核机制保证信息质量。

## 项目背景与目标

华中农业大学校园体量大、场景多，新生和部分在校生很难快速掌握高质量校园信息（例如食堂档口性价比、周末适合活动的地点、教学楼设施体验等）。  
本项目旨在构建一个聚焦校内真实体验的评分平台，提升校园信息流通效率，同时为校园管理和优化提供反馈参考。

## 目标用户

- 在校本科生（尤其新生）
- 研究生
- 教师
- 对校园感兴趣的校外访问者（轻量浏览）

## 当前实现（MVP）

- 邀请码注册/登录（Cookie 鉴权）
- 创建榜单（支持新增对象 + 选择可复用对象，总数至少 3 个）
- 管理员审核 Board/BoardItem（通过/驳回）
- 前台仅展示审核通过内容
- 对 BoardItem 评分（1.0~5.0，0.5 步长）
- 同用户对同对象评分覆盖更新
- 基础管理页与用户页分离

## 技术栈

- Frontend: Next.js (App Router) + TypeScript
- Backend: Next.js API Routes
- Database: PostgreSQL (Neon)
- ORM: Prisma
- Auth: JWT + HttpOnly Cookie

## 核心业务规则

- `BoardItem` 必填字段：`name`、`description`、`imageUrl`
- 创建榜单时对象总数（新增 + 复用）必须 `>= 3`
- Board / BoardItem 必须审核通过后才在前台可见
- Review 不需要审核，但只允许对可见对象评分
- 同一用户对同一对象只能保留一条评分（后续覆盖更新）

## 快速启动

1. 配置环境变量

```bash
cp .env.example .env.local
cp .env.local .env
```

2. 安装依赖

```bash
npm install
```

3. 同步数据库结构并导入示例数据

```bash
npx prisma db push
npm run prisma:seed
```

4. 启动开发环境

```bash
npm run dev
```
